Stage 0 - Important Concepts

Vocab

Tag-The markup part of HTML, what the code looks like
Element-The name for an entire line of code -- Open tag, content, closing tag
Attribute- A feature of tags that allows you to add links, images etc.

HTML

-HyperText Markup Language
-4 parts are text content, markup, references,and links

Attributes

- Allows for things to added to a page, like links or images ( < a> and < img>)
- Use values to enter in the URL of the source and then content is what is seen and clicked on to follow links
- Images are void tags -- no end tag or content

White Space
- Must use tags to insert
- < br> : inline
Simply telling the text to break off and go down to the next line
- < p> : block
Puts an invisible box around text and allows it to be edited as a single unit
